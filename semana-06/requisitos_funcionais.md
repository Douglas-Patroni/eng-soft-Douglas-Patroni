RF01 - O sistema deve disponibilizar uma página de vendas para exibir os produtos da associação, permitindo a compra direta pelos usuários.

RF02 - O sistema deve apresentar uma página com um mapa interativo indicando a localização dos produtores associados.

RF03 - O sistema deve permitir que a associação edite os produtos da área de vendas, incluindo a remoção dos esgotados.

RF04 - O sistema deve disponibilizar um campo para cadastro de email, permitindo o envio de notificações para usuários e produtores.

RF05 - O sistema deve possuir uma página de login para identificação de usuários, produtores e administradores para personalização e acesso a funcionalidades específicas.

RF06 - O sistema deve permitir que produtores incluam novos produtos no catálogo, mantendo o mesmo atualizado.

RF07 - O sistema deve enviar email ao produtor com informações sobre as vizualizações de seus produtos e também indicar os mais vistos.
