# Site para divulgação e comércio para uma associação de produtores de café

1. Como associação, quero que o site tenha uma página de venda, para fortalecer nossas vendas.

2. Como associação, desejo que o site possua uma página com o mapa dos produtores, para identificar onde são produzidos.

3. Como associação, gostaria de editar os produtos da área de venda, para retirar produtos esgotados.

4. Como associação, quero que tenha um campo de e-mail, para que os usuários e produtores recebam notificações personalizadas.

5. Como associação, desejo que possua uma página de login, para facilitar a navegação e personalização do site.

6. Como produtor, quero que o sistema atualize o catálogo de produtos com produtos novos, para que esteja sempre atualizado.

7. Como produtor, gostaria de receber notificação por e-mail de quantas pessoas visualizaram meus produtos e quais foram os mais vistos, para ter o controle de quais geram mais interesse.

8. Como produtor, gostaria de poder ver as avaliações dos meus produtos, para saber se estão agradando os clientes.

9. Como usuário, desejo que possua um campo de texto, para poder comentar sobre o produto que comprei.

10. Como usuário, gostaria de filtrar os produtos por marca ou tipo, para encontrar o que desejo mais facilmente.

11. Como usuário, quero que tenha contato por e-mail ou telefone, para resposta de dúvidas rápidas.

12. Como usuário, desejo que possua a opção de cartão ou pix, para facilitar o pagamento.
